# TinyApp Project

## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session
- morgan

## Getting Started

- Install all dependencies (using the `npm install` command).

```
npm install express 
npm install bcryptjs
npm install cookie-session
npm install morgan
```

- Run the development web server using the `node express_server.js` command.
